About

Это мой проект, который я, постепенно, дорабатываю по мере моих возможностей. Некоторые разделы, такие как, Blog и Portfolio еще не готовы, потому что я не нашел дизайн\не придумал, что там будет.
Весь код можно почитать на страничке этого проекта github

В проекте были использованы следующие технологии:

  • HTML5
  • CSS3
  • Все стили были прописаны с помощью SCSS
  • SCSS конвертировался автоматически в css с помощью таск-менеджера Gulp
  • Комментарии в article работают на чистом JavaScript
  • В футере сайта есть спойлеры написанные с помощью jQuery
